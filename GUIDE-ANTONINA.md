# Guide d'utilisation du site — Antonina

Ce guide t'explique comment mettre à jour le site `chatterie-sciortino.netlify.app` toi-même, sans aide technique.

## Pour te connecter

1. Va sur **https://chatterie-sciortino.netlify.app/admin**
2. Clique sur **"Login with GitHub"**
3. Connecte-toi avec ton compte GitHub (le compte que tu as créé pour le site).

Tu arrives sur l'interface d'administration, en français.

> 💡 Tu peux te connecter depuis ton téléphone — le site fonctionne aussi bien sur mobile.

## Ajouter un nouveau chat (parent)

1. Clique sur **"Nos chats"** dans le menu de gauche.
2. Clique sur **"+ Nouveau chat"**.
3. Remplis :
   - **Nom** (ex : Luna)
   - **Sexe** (mâle ou femelle)
   - **Date de naissance**
   - **Couleur / Pattern** (ex : "Seal point mitted")
   - **Photo principale** : glisse une photo depuis ton téléphone ou ton ordinateur
   - **Galerie** : ajoute autant de photos que tu veux
   - **Description** (facultatif) : un petit texte pour présenter le chat, son caractère
4. Clique sur **"Publier"** en haut à droite.

Le site se met à jour automatiquement en moins d'une minute.

## Ajouter une nouvelle portée

1. Clique sur **"Portées"**.
2. Clique sur **"+ Nouvelle portée"**.
3. Remplis :
   - **Titre** (ex : "Portée 2026 — Luna × Apollon")
   - **Date de naissance** des chatons
   - **Père** et **Mère** (à choisir parmi les chats déjà créés)
   - **Numéro de portée LOOF** (obligatoire — donné par le LOOF lors de la déclaration)
   - **Numéro ICAD de la mère** (obligatoire — sur la carte d'identification)
   - **Photo de couverture** (facultatif)
4. Pour chaque chaton, clique sur **"+ Ajouter un chaton"** et remplis :
   - Nom, sexe, couleur
   - **Statut** : disponible / réservé / parti
   - Photos
5. Clique sur **"Publier"**.

> ⚠️ Les numéros LOOF et ICAD sont obligatoires pour la loi française. Sans
> eux, l'annonce des chatons sur le site ne serait pas légale.

## Mettre à jour le statut d'un chaton

1. Va dans **"Portées"** → ouvre la portée concernée.
2. Trouve le chaton dans la liste.
3. Change son statut (disponible → réservé → parti).
4. Clique sur **"Publier"**.

## Modifier les pages "Démarche", "Contact", "Mentions légales"

1. Clique sur **"Pages"** dans le menu.
2. Choisis la page à modifier.
3. Modifie le texte directement.
4. Clique sur **"Publier"**.

## Si quelque chose ne marche pas

- **Tu n'arrives pas à te connecter** : assure-toi d'utiliser le bon compte GitHub (celui qui a accès au repo).
- **Le site ne se met pas à jour** : attends 1-2 minutes après "Publier". Sinon, écris à Merlin.
- **Tu as supprimé quelque chose par erreur** : pas de panique — tout l'historique est conservé sur GitHub. Merlin peut tout récupérer.

## Conseils pour les photos

- **Format** : JPEG ou PNG, idéalement 1500 à 2500 pixels de large.
- **Poids** : pas besoin de réduire toi-même, le site optimise tout automatiquement.
- **Lumière** : les photos prises près d'une fenêtre, à la lumière du jour, sont toujours plus belles.
- **Cadrage** : une photo carrée ou en format portrait passe bien sur la fiche d'un chat.

Bonne mise en ligne ! ✨
